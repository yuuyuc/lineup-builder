# lineup-builder
サッカーのスタメン・フォーメーションを視覚的に組み立てられる無料のWebアプリです。
インストール不要、ブラウザだけで動作します。

---

## 🎯 用途

- 試合前のスタメン・フォーメーションを整理したい
- 対戦チームとのマッチアップを並べて確認したい
- ラインナップ画像をSNSにシェアしたい

---

## 📖 使い方

1. **フォーメーション選択**：画面上部のドロップダウンから選択（22種類対応）
2. **選手を配置**：左パネルでリーグ・チームを選択して選手を読み込み、ドラッグ＆ドロップで配置
3. **対戦相手設定**：「⚔️ 対戦相手」ボタンでアウェイチームを追加（横並び表示）
4. **保存・シェア**：画像として保存、またはURLをコピーしてシェア

### 対応リーグ
| リーグ | データ取得元 |
|--------|------------|
| Jリーグ | Wikipedia |
| プレミア・ラ・リーガ・ブンデス・セリエA・リーグアン | ESPN |
| 日本・ブラジル・フランスなど17カ国代表 | 内蔵プリセット |

---

## ⚠️ 注意点

- **TEST版**：お試し公開です。Jリーグや国代表はチーム情報、選手情報が古いです。5大リーグのクラブも情報が反映されていないクラブが多数あります
- **Jリーグ選手写真**：Wikipedia取得のため、一部の選手は写真なし（イニシャル表示）になります
- **画像保存**：写真の読み込み完了後に保存してください（読み込み中は黒丸になる場合があります）
- **スマホ推奨ブラウザ**：Chrome / Safari
